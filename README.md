## Execution
1. Clone the repository
```
   git clone https://github.com/anant-357/DBMS_IIITG_Project.git
   cd DBMS_IIITG_Project
```
   
2. Source
``` 
    source .RealEstateWebsite-env/bin/activate 
```
3. Run
``` 
    python main.py 
```
